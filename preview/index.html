QUICK START

5 分钟接入，
无需改业务代码

用官方 OpenAI SDK，改一个 base_url 就能享受全部厂商。 切换底层模型 / 遇到故障自动切换 / 按 Token 精确计费，全部交给 LLMHub。

  • 1注册并创建 API Key
  • 2充值（或使用新用户额度）
  • 3改一行代码 —— base_url = https://api.llmhub.io/v1
  • 4开始调用
from openai import OpenAI

client = OpenAI(
    api_key="sk-llmh-xxxxxxxxxxxx",
    base_url="https://api.llmhub.io/v1",
)

resp = client.chat.completions.create(
    model="gpt-4o-class",
    messages=[
        {"role": "user", "content": "你好 LLMHub"}
    ],
    stream=True,
)

for chunk in resp:
    print(chunk.choices[0].delta.content or "", end="")
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-llmh-xxxxxxxxxxxx',
  baseURL: 'https://api.llmhub.io/v1',
});

const stream = await client.chat.completions.create({
  model: 'gpt-4o-class',
  messages: [{ role: 'user', content: '你好 LLMHub' }],
  stream: true,
});

for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0]?.delta?.content ?? '');
}
